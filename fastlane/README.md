# Fastlane Metadata

These files are draft App Store metadata for manual review or future
Fastlane use.

- Replace TODO URLs and contact details before submission.
- Do not commit App Store Connect API keys, credentials, certificates,
  provisioning profiles, session files, or private tokens.
- This repo does not need a Fastfile just to store metadata.
- If Fastlane is later used for screenshots or upload, verify current
  Fastlane screenshot size support before upload.
