# App Store Release Package

This folder contains draft App Store release material for `Report Comment Writer`.
It was integrated into the live `CommenterIOS` checkout and should be
checked again against the final submitted binary before App Store
submission.

## Included

- App Store Connect copy, privacy answers, review notes, age rating notes, and remaining inputs.
- Privacy, terms, open-source notices, and school-policy drafts.
- Brand, teacher-language, trust-boundary, FAQ, landing-page, press, and launch copy.
- App icon, logo, social, and draft 6.9-inch screenshot masters.
- Support site draft HTML pages with no analytics or external scripts.
- Repo evidence, source/policy notes, release risks, validation scripts, and Fastlane metadata.

## Before Submission

Replace every `TODO:` placeholder, host the public support and privacy
pages, capture final screenshots from the real app UI, re-check privacy
answers against the final binary, and confirm App Store Connect fields.
