# Report Comment Writer

Turn teacher notes into clear report comment drafts. Works without Wi-Fi, needs no sign-in, and every comment is checked by you before export.

Start with your notes. Finish with a clear report comment.

Built for Australian primary teachers writing Year 5 and Year 6 report
comments. Create a class, add students, enter achievement information,
prepare draft wording, check every comment, then save or share when ready.
