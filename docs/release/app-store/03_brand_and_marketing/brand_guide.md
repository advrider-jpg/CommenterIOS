# Brand Guide

App Store name: Report Comment Writer

Home screen name: Report Writer

Tagline: Start with your notes. Finish with a clear report comment.

Visual direction:

- A sheet of paper for reports.
- A comment bubble for report comments.
- A checkmark for teacher review.
- Warm paper colours.
- Plain, calm, teacher-friendly.
- No mascot, cartoon child, school logo, department seal, or text inside the iOS icon.

Colours:

- Paper background: #F6EFDF
- Paper surface: #FFFAEC
- Deep paper: #F4E8D3
- Ink: #171411
- Muted ink: #756D61
- Action blue: #256DC8
- Soft blue: #E6EEF8
- Teacher green: #3B8750
- Soft green: #E4F1DD
- Attention orange: #D8792C
- Soft orange: #FAE7D2
- Gold accent: #D7A321
