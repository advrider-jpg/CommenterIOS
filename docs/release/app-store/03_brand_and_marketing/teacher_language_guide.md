# Teacher Language Guide

Use plain Australian school language:

- report comment
- student report
- class list
- teacher notes
- achievement level
- achievement and progress
- strengths
- areas for improvement
- next steps
- evidence from class work
- parent-friendly wording
- check every comment
- save or share
- works without Wi-Fi
- no sign-in
- online account
- teacher judgement

Avoid hype, consultant language, and claims that make the app sound like
it replaces teacher checking.
