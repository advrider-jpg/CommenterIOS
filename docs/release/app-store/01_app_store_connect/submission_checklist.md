# Submission checklist

## App Store Connect

- [ ] App Store name: `Report Comment Writer`.
- [ ] Subtitle: `For Australian teachers`.
- [ ] Category: Education.
- [ ] Bundle ID matches Xcode: `com.commenterios.app`.
- [ ] SKU: `report-comment-writer-ios-001` or your preferred internal SKU.
- [ ] Copyright owner added.
- [ ] Support URL live.
- [ ] Privacy Policy URL live.
- [ ] App Review contact name, email and phone added.
- [ ] No demo account entered because there is no sign-in.
- [ ] App privacy answers completed.
- [ ] Age rating completed.
- [ ] Pricing and territories selected.
- [ ] Version number selected. Recommended public version: `1.0`.

## Build and app assets

- [ ] App icon asset catalog added to the app target.
- [ ] Marketing icon present at 1024 × 1024 px.
- [ ] Display name remains `Report Writer` or final short name.
- [ ] App has no unintended analytics, tracking, crash reporting, remote AI, advertising, subscriptions or server calls.
- [ ] Privacy manifest still matches the final binary.
- [ ] App opens and works in airplane mode.
- [ ] User-facing copy avoids “local-only”, “offline-first”, and other jargon.

## Screenshots

- [ ] Real simulator/device screenshots captured from the release build.
- [ ] Screenshots use sample or non-identifying student data only.
- [ ] Screenshots do not include third-party logos, real student names, prices, unsupported claims or misleading UI.
- [ ] At least one and no more than ten screenshots uploaded for the required iPhone size.
- [ ] Captions match the real screen shown.

## Legal/support

- [ ] Privacy policy published.
- [ ] Support page published.
- [ ] Terms/EULA page published or Apple standard EULA selected.
- [ ] Open-source notices prepared and linked from support or in-app support.
- [ ] School-policy disclaimer included.

## Final copy audit

- [ ] No hard “Australian Curriculum aligned” claim unless separately verified.
- [ ] No “approved by schools/departments” claim.
- [ ] No “student data never leaves your phone” claim.
- [ ] No “AI writes reports” claim.
- [ ] No claim that the app works for all year levels or every subject.
