## Description

Report Comment Writer helps Australian teachers write Year 5 and Year 6 student report comment drafts without starting from a blank page.

Add your class, choose the learning areas you are reporting on, enter each student’s achievement level, evidence and next step, then create a draft comment.

Every comment can be edited. Nothing is final until you have checked it yourself.

Designed for the language teachers already use:

- achievement and progress
- strengths
- areas for improvement
- next learning steps
- evidence from class work
- parent-friendly wording

Useful at report time:

- create a class
- add students by hand
- import class lists and results from CSV, XLSX or XLS files
- write comments for English, Mathematics, Science, HASS, Health and Physical Education, The Arts and Technologies
- review and edit every comment
- prepare DOCX report files
- prepare XLSX or XLS review workbooks
- back up and restore a class file

No sign-in. Works without Wi‑Fi. The app does not upload student names or report notes to an online account. You choose when to save, export or share a file.

Report Comment Writer does not replace teacher judgement. It helps you prepare a draft, check the wording, and keep control of the final report comment.
