# App Review Notes

No sign-in is required. No account is required. No demo account is
needed.

Core report-comment work does not require internet access. Reviewers can
test with fake sample data.

Suggested test path:

1. Open the app.
2. Create a class named `Room 5`.
3. Add a fake student named `Ava Ng` in Year 5.
4. Select English.
5. Add achievement information: `At Standard`.
6. Add evidence: `Uses evidence from the text to explain ideas.`
7. Add next step: `Add more detail when comparing texts.`
8. Create a draft comment.
9. Review and edit the draft.
10. Check the comment before export.
11. Prepare a DOCX report or XLSX/XLS review workbook.
12. Use the native save or share sheet if it appears.

The teacher checks every comment before export. If a native save/share
sheet is cancelled, the app should show cancellation rather than success.

On-device writing suggestions may be unavailable in the review
environment. Deterministic draft generation remains the baseline path.
