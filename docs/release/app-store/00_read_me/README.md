# Report Comment Writer release package

Prepared: 2026-06-08

This package contains the App Store release material that can be prepared without your Apple account details and without running a final App Store build. It is grounded in the uploaded repo, Australian teacher-reporting language, and the decision to call the app **Report Comment Writer**.

## What is included

- App Store Connect copy and field-by-field metadata.
- App Review notes and a reviewer test path.
- App privacy answers, age-rating assumptions, and release checklist.
- Privacy policy, support page, FAQ, and simple website copy.
- Brand guide, teacher-language guide, and trust-boundary guide.
- Generated logo, app icon, social card, landing hero image, colour swatches, and draft App Store screenshot masters.
- Xcode app icon asset catalog files.
- Repo evidence notes explaining which claims are safe and which claims are not.

## What is not final until you add details

The package uses placeholders for public contact details, privacy policy URL, support URL, App Review phone/email, seller name, copyright owner, price, territories, and release timing. Search for `TODO:` across the package.

## Important release stance

Use simple teacher language:

> No sign-in. Works without Wi‑Fi. The app does not upload student names or report notes to an online account. You choose when to save, export or share a file.

Do not lead with AI. Do not say “local-only”. Do not say “Australian Curriculum aligned” unless the app content is separately verified against the exact curriculum and states you want to claim.

## Generated image assets

The generated icon is a simple paper, comment, and checkmark mark. It avoids text inside the icon because small text is hard to read and can make app icons cluttered. The generated screenshot images are **draft marketing masters**, not final simulator screenshots. They should be compared against real app screenshots from the repo’s screenshot UI test workflow before submission.
